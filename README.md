# System Requirements

* Windows PowerShell 5.1 and .NET 4.6.1+
* PowerShell Core 6+

# Installing

To install globally:

```powershell
Install-Module -Name 'W3CLogs'
Import-Module -Name 'W3CLogs'
```

To install privately:

```powershell
Save-Module -Name 'W3CLogs' -Path '.'
Import-Module -Name '.\W3CLogs'
```
